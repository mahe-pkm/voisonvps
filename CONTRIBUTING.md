# Contributing to Vois

Thank you for your interest in contributing to Vois!

## Getting Started

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork locally.
3.  **Create a Branch** for your feature or fix (`git checkout -b feature/amazing-feature`).

## Guidelines

- **Code Style**: We use Prettier and ESLint. Please ensure your code is formatted before committing.
- **Commits**: Use clear, descriptive commit messages.
- **Pull Requests**: Explain what your PR does and link to any relevant issues.

## Reporting Issues

If you find a bug, please open an issue describing:
1.  Steps to reproduce.
2.  Expected behavior vs. actual behavior.
3.  Screenshots (if applicable).

Thank you for making Vois better!
