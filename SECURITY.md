# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Vois, please do NOT create a public GitHub issue.

Instead, please send an email to **support@vois.admin** (or your security contact). We will review the issue and respond within 48 hours.
